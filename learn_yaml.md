# YAML SYNTAXES
- YAML :- Yaml Ain't Markup Language
- File Extension Should be .yml or .yaml
- EX:
   ```
     config-1
     ---
     config-2
   ```

# YAML Data Types
## 1 . Scalar Data Types
- Numbers: It Can be **Integer** and **Floating** values
    - Ex: 23, 23.5
- Boolean: The Value Can **True** or **False**
    -Ex: is_student: true/false
